Heroku 網站

1.進入 https://heroku.com,

使用 @gm 登記一個 Heroku 帳號, 關鍵在於密碼設定必須要至少 8 個字元, 且字元中必須包含符號, 英文字母及數字, 登記後要進入 @gm 帳號進行確認後才能開通 Heroku

2.建立一個新的倉儲如:s409231xx

3.下載Heroku cli 程式解開至隨身碟

4.修改start路徑:

set path_heroku=%Disk%:\heroku\bin;
path=%Disk%:;%path_python%;%path_portablegit% ;%path_heroku% ;%path%

輸入:heroku version 

有回應即可成功

6.heroku login -i, 完成後產生 y:\home_ipv6\_netrc

7.設定同步倉儲  heroku git:remote -a s409231xx

8.正常推送git push heroku
